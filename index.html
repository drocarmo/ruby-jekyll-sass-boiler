Wheat is a simple Jekyll template for designers.

It is a great starting point to help any designer quickly wip out a simple content-managed app, a product marketing page, prototype an idea or any instance of building a dynamic site.

Everything in Wheat was built to be extremely watered down for quick and iterative design in the browser. {% for post in site.posts %} {{ post.title }}{% endfor %}.

Clone the github repository on your local machine and start building:

git clone https://github.com/drocarmo/wheat.git
